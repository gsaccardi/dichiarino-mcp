What is Dichiarino?

Dichiarino is an MCP (Model Context Protocol) server that embeds official Italian fiscal rules, IRPEF rates, and deduction limits. Connect it to Claude Desktop, Cursor, or any MCP client and your AI assistant gains 12 specialized tools to guide you through your annual 730 tax return.

The Agenzia delle Entrate provides no public API for the 730 precompilato. Dichiarino works as a knowledge + calculation engine — it cannot submit your return for you. Always verify results with a qualified professional (CAF, commercialista).

12 MCP Tools

Every tool your AI assistant needs to help with the Italian 730 form.

calcola_irpef
IRPEF lorda from income + year (2024: 23% / 35% / 43%)
calcola_detrazione_lavoro
Work income tax credit — formula per bracket + €65 bonus
calcola_detrazioni_familiari
Detrazioni for spouse, children ≥21, other dependants
calcola_oneri
Quadro E deductions — medical, mortgage, renovation, university…
verifica_spesa_detraibile
Is this expense deductible? Which quadro? How much?
valida_codice_fiscale
Validate + parse Italian codice fiscale (check digit algorithm)
calcola_risultato_dichiarazione
Full 730 result — rimborso (refund) or debito (tax due)
guida_quadro
Step-by-step guide for any Quadro (A–W, M, T)
lista_documenti_spesa
Documents needed for a specific expense type
genera_checklist_730
Personalised compilation checklist based on your situation
calcola_addizionale_regionale
Regional IRPEF surcharge for all 20 Italian regions
analizza_certificazione_unica
Validate CU (Certificazione Unica) fields for consistency

Quick Start

Up and running in under a minute.

Step 1 — Clone & install
bash 
git clone https://github.com/gsaccardi/dichiarino-mcp.git
cd dichiarino-mcp
uv sync
Step 2 — Add to Claude Desktop
json 
{
  "mcpServers": {
    "dichiarino": {
      "command": "uv",
      "args": [
        "--directory",
        "/path/to/dichiarino-mcp",
        "run",
        "dichiarino"
      ]
    }
  }
}
Step 3 — Ask away
prompt 
Calcola l'IRPEF per un reddito di 35.000€ nel 2024.

See It In Action

A real interaction between a user and an AI assistant with Dichiarino connected.

Claude Desktop
You: Ho speso 1.200€ dal medico quest'anno. Cosa posso detrarre?
Claude: calcola_oneri
Spese sanitarie: 1.200€ − franchigia 129,11€ = 1.070,89€
Detrazione IRPEF: 1.070,89€ × 19% = 203,47€
Documenti: fatture mediche, scontrini farmacia con CF
