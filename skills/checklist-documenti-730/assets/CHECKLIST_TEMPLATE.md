# Checklist Documenti Modello 730

**Anno fiscale:** ___________  
**Contribuente:** ___________  
**Codice fiscale:** ___________

---

## ✅ Documenti obbligatori per tutti

- [ ] Documento d'identità valido
- [ ] Tessera sanitaria / codice fiscale
- [ ] IBAN del conto corrente (per rimborso)
- [ ] 730 / Redditi dell'anno precedente (se disponibile)

---

## 📋 Redditi

- [ ] **CU (Certificazione Unica)** dal datore di lavoro o INPS
- [ ] CU da altri committenti (co.co.co., collaborazioni)
- [ ] Ricevute compensi occasionali (< €5.000)
- [ ] Documentazione redditi da lavoro autonomo

---

## 🏠 Immobili

- [ ] Visura catastale (rendita catastale aggiornata)
- [ ] F24 IMU (se seconda casa o non esente)
- [ ] Contratti d'affitto (se affittato)
- [ ] Contratto cedolare secca (se optato)

---

## 🏦 Mutuo

- [ ] Attestato interessi passivi (dalla banca, anno corrente)
- [ ] Atto di mutuo (prima stipula)
- [ ] Rogito notarile (prima casa)

---

## 🏥 Spese sanitarie

- [ ] Scontrini parlanti farmacia (con CF)
- [ ] Fatture medici / specialisti / dentisti
- [ ] Fatture analisi cliniche, radiologie
- [ ] Fatture per occhiali / lenti (con prescrizione)
- [ ] Fatture protesi e dispositivi medici
- [ ] Fatture per spese veterinarie

---

## 👨‍👩‍👧 Famiglia

- [ ] Codici fiscali familiari a carico
- [ ] Dichiarazione di essere a carico (figli > 18 anni)
- [ ] Ricevute asilo nido (con CF del bambino)
- [ ] Ricevute attività sportiva ragazzi 5–18 anni

---

## 📚 Istruzione

- [ ] Ricevute tasse universitarie
- [ ] Ricevute rette scolastiche (materna → superiori)
- [ ] Attestazioni università privata (limite MIUR)

---

## 🚌 Trasporti e altri oneri

- [ ] Abbonamento trasporti pubblici (annuale/mensile)
- [ ] Premi assicurazione vita / infortuni
- [ ] Ricevute spese funebri
- [ ] Attestazioni donazioni (bonifico + ricevuta ente)

---

## 💼 Previdenza e contributi

- [ ] Contributi INPS artigiani/commercianti (se autonomo)
- [ ] Estratto conto fondo pensione integrativo
- [ ] F24 versamenti volontari INPS

---

## 🌍 Estero (se applicabile)

- [ ] Estratti conto conti esteri
- [ ] Attestazioni partecipazioni / investimenti esteri
- [ ] Certificazione imposte pagate all'estero

---

*Conservare tutti i documenti per **5 anni** dalla presentazione della dichiarazione.*  
*Fonte: art. 43 DPR 600/1973 - termini di accertamento.*
