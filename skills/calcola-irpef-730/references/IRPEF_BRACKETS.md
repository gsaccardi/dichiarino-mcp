# IRPEF Brackets - Reference

Official Italian IRPEF tax brackets for years 2023, 2024, and 2025.

---

## 2025 brackets (identical to 2024 - permanent reform, L. 111/2023)

| Scaglione | Aliquota |
|-----------|----------|
| €0 – €28.000 | 23% |
| €28.001 – €50.000 | 35% |
| oltre €50.000 | 43% |

## 2024 brackets (same as 2025)

See table above.

## 2023 brackets

| Scaglione | Aliquota |
|-----------|----------|
| €0 – €15.000 | 23% |
| €15.001 – €28.000 | 25% |
| €28.001 – €50.000 | 35% |
| oltre €50.000 | 43% |

---

## Detrazioni lavoro dipendente 2024/2025

| Reddito complessivo | Detrazione |
|---------------------|-----------|
| ≤ €15.000 | 1.955 × (reddito / 15.000) - min €690 (€1.380 se reddito ≤ €8.000) |
| €15.001 – €28.000 | 1.910 + 1.190 × (28.000 − reddito) / 13.000 |
| €28.001 – €50.000 | 1.910 × (50.000 − reddito) / 22.000 |
| > €50.000 | 0 |

## Taglio cuneo fiscale 2025 (L. 207/2024)

Applied **on top of** the standard detrazione lavoro:

| Reddito | Beneficio aggiuntivo |
|---------|---------------------|
| ≤ €8.500 | 7,1% del reddito |
| €8.501 – €15.000 | 5,3% della quota eccedente €8.500 |
| €15.001 – €20.000 | 4,8% della quota eccedente €15.000 |
| €20.001 – €32.000 | €1.000 fisso |
| €32.001 – €40.000 | €1.000 × (40.000 − reddito) / 8.000 |
| > €40.000 | 0 |

## Detrazioni pensione 2024/2025

| Reddito complessivo | Detrazione |
|---------------------|-----------|
| ≤ €8.500 | 1.955 (minimo garantito €713.50) |
| €8.501 – €15.000 | 700 + 1.255 × (15.000 − reddito) / 6.500 |
| €15.001 – €55.000 | 700 × (55.000 − reddito) / 40.000 |
| > €55.000 | 0 |

---

## No-tax area (2025)

- Lavoro dipendente: reddito ≤ **€8.500** (no IRPEF netta after cuneo)
- Pensione: reddito ≤ **€8.500**
- Lavoro autonomo: reddito ≤ **€5.500**
